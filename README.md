# AI Builders Class

A single-page landing site for an AI class that teaches entrepreneurs how to automate
their business using GenSpark, Claude, and ChatGPT.

## Features

- Live classes Tuesday through Friday
- Open community of 140+ members
- Single call-to-action to join

## Run locally

Just open `index.html` in your browser — no build step required.

```bash
open index.html
```
